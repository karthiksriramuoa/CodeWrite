C Programming

Start date:October 2 50 students registered for this course

You will learn a high-level and general-purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed at Bell Labs by Dennis Ritchie.

C++

Start date:October 2 30 students registered for this course

You will learn a general-purpose object-oriented programming (OOP) language, developed by Bjarne Stroustrup, which is an extension of the C language. It is therefore possible to code C++ in a "C style" or "object-oriented style."

Java

Start date:October 2 35 students registered for this course

You will learn a computer programming language that is class-based, object-oriented, and is designed to have as few implementation dependencies as possible and can be used to create applications that can run on a single/distributed system.
